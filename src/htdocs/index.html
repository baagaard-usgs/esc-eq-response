Earthquake Response App

This web application creates a map and summary to help collect and digest information used in talking points and scientific response.

Instructions

  1. Enter the Mainshock Event ID

    Select an earthquake from ‘Significant Earthquakes in the Past Month’ or enter an Event ID manually.

    You can find an Event ID in the URL of an Event Page for a specific earthquake. For example:

    Find an Earthquake:

    • Latest Earthquakes
    • Search Earthquake Archives
  2. Tweak the parameters for Aftershocks / Historical Seismicity (optional)

    Initial values will be set automatically based on the mainshock’s magnitude and estimated rupture length.

  3. Select the ‘Map’ or ‘Summary’ tab
